Trainvial

Este proyecto contiene una aplicación construída con AngularJS para probar y conocer su funcionamiento. Se trata de un juego de tablero de preguntas y respuestas. Se usan aspectos básicos de Angular como módulos, controladores y filtros.

También utilicé Bootstrap 2 para conocer más este framework no solo de forma teórica. Del mismo modo incorporé algunas animaciones CSS3.

Apliqué responsive design y probé la aplicación en diferentes dispositivos móviles.

En definitiva este pequeño proyecto fue el trampolín que me ayudó a desenvolverme mejor en futuros proyectos.
