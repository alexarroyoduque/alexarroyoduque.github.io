amiibum

Mi primer experimento con lit-html y lit-element la evolución de Polymer.

El sitio web sirve como álbum para marcar los amiibos de Nintendo coleccionados por el usuario. Permite filtrar por serie y/o personaje y almacena los datos localmente.

Los datos son propocionados por una api creada por otro usuario. Sin embargo al estar servida con firebase que solo permite llamadas https y la api está en http, los datos son almacenados localmente en el proyecto.

Esto que podría ser una desventaja sirve para mejorar la carga de datos ya que se descargan muchas imágenes que si no son reducidas pensan demasiado.obstante los componentes para llamar a la api están implementados.
