## Quick start

```sh
$ git clone https://github.com/alexarroyoduque/alexarroyoduque.github.io.git
$ cd alexarroyoduque.github.io
$ npm install
$ npm install -g polymer-cli
$ polymer serve

```

## Build

```sh
$ polymer build
$ polymer serve build/default
$ npm run deploy
```
