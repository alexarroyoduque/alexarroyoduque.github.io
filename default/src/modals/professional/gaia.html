Colaboración en la creación del framework GAIA para una importante empresa española del sector de seguros con presencia en multitud de países.

  • Objetivo: Con el framework GAIA se agilizará el proceso de desarrollo para crear aplicaciones web basado. Las aplicaciones serán construidas por los desarrolladores con tecnologías como AngularJS y JADE.

  • Tareas: Ayudar en todo lo que fuera necesario. Sobre todo centrando las tareas en la resolución de incidencias y en la creación de nuevas operativas.

  • Metodología: SCRUM.

  • Tecnologías: Yeoman, AngularJS, NodeJS, JADE, HTML y CSS.
